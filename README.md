# Critica
A custom WordPress theme for Rule29 and Critica.life.

## Getting Started
To get started, clone this repo and run `npm install && bower install`

## Packages Used
- Node/npm
- Gulp
- Ginger by Eric Stout
- Nifty Nav by Factor1
- Sugar by Factor1
- Slick Carousel by Ken Wheeler
